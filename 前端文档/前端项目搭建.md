## 技术栈

vue3    typescript   ArcoDesign 

## 安装

**初始化vue项目**

需要添加的项有：`vue3`、`TypeScript`、`pinia`、`router`、`store`

```shell
npm init vue@latest
```

**安装arto design、mock js、axios**

```shell
npm install --save-dev @arco-design/web-vue
npm i axios
npm i mockjs
npm i @types/mockjs
npm install sass --save-dev
```
